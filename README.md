# 写给悦宝宝的一封信

一个部署在 GitHub Pages 上的沉浸式信件网页。

## 页面体验

- 点击信封后播放翻盖、抽出信纸和场景淡入动画
- 页面内容自动从 `content.md` 读取
- 使用 Markdown 续写章节，无需修改 HTML
- 适配桌面端与移动端
- 支持系统“减少动态效果”设置

## 编辑信件

只需打开 `content.md` 修改文字：

```markdown
# 悦宝宝

## 我记得的你

在这里继续写新的内容。
```

- `#` 后面是收信人的称呼
- `##` 创建一个新章节
- `###` 创建并列的小卡片
- `>` 创建居中的引用文字
- `---` 后面填写信件落款

保存并推送 `content.md` 后，网页会自动显示新内容。

## 目录结构

```text
.
├─ content.md
├─ index.html
└─ assets
   ├─ css
   │  └─ style.css
   └─ js
      └─ main.js
```

## 本地预览

在项目根目录运行：

```powershell
python -m http.server 8765
```

然后访问 <http://127.0.0.1:8765/>。
